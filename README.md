🎨 Modern Color Palette Generator with Lock Colors, Gradient Generator, Local Storage, PNG Export, and Responsive UI using HTML, CSS, and JavaScript.

# 🎨 Color Palette Pro

A modern and interactive Color Palette Generator built using HTML, CSS, and JavaScript.

Generate beautiful color palettes, lock your favorite colors, save palettes locally, create gradients, and export palettes as PNG images.

---

## 🚀 Features

### 🎲 Random Color Palette Generator
Generate a new set of colors instantly with a single click.

### 📋 Copy HEX Codes
Click on any HEX code to copy it to the clipboard.

### 🔒 Lock Colors
Lock individual colors while generating a new palette.

### ❤️ Save Favorite Palettes
Save palettes to Local Storage and access them later.

### 🌈 Gradient Generator
Create gradients using generated colors and copy the CSS code.

### 📸 Export as PNG
Download the generated palette as an image.

### ⌨️ Keyboard Shortcut
Press **Spacebar** to generate a new palette quickly.

### 💾 Local Storage Support
Saved palettes remain available even after refreshing the page.

## 🎯 Usage

### Generate Palette

Click:

```text
Generate Palette
```

or press:

```text
Space
```

### Lock Colors

Click the lock icon:

```text
🔓 → 🔒
```

Locked colors remain unchanged during generation.

### Copy HEX Code

Click any HEX code.

Example:

```text
#4F46E5
```

### Save Palette

Click:

```text
❤️ Save Palette
```

Saved palettes appear below the generator.

### Export PNG

Click:

```text
📸 Export PNG
```

to download the palette as an image.
